Installing Circuit virtual machine

Prerequisites

  • The pir and json modules present, built and installed.
  • All prerequisites of the pir module. See the pir INSTALL file.
  • bnfc version 2.3b or later, to compile the C-Json library. Need version 2.3b as the artifact layout (ie. what C, Bison etc. files are prodiuced) has changed since 2.2.
  • modules pir and json: present, built and installed. Their installed locations are specified in config.make and can be changed.

    Configuration

    The main configuration decision is whether the TTP modules will run on a 4758, or on a normal host. Please see the pir INSTALL file.

    Building

    $ make
$ make install
    Installs the executables and shared objects (if used) into DIST_ROOT, which is specified in config.make.
