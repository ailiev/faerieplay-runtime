Installing Circuit virtual machine

Prerequisites

  • The pir and json modules present, built and installed, with prerequisites. Their source locations are specified in config.make and can be changed.

    • Configuration

    The main configuration decision is whether the TTP modules will run on a 4758, or on a normal host. Please see the pir INSTALL file.

    Building

    $ make dep install
    Installs the executables and shared objects (if used) into directory DIST_ROOT, which is specified in config.make.
